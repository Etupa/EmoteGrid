# EmoteGrid

A standalone Dalamud plugin providing a scalable and customizable ImGui grid for emotes.

## Commands
- `/emotegrid`: Open the main Emote Grid window.
- `/emotegrid refresh`: Manually refresh the emote list from the game data.

## Features
- **Scalable Grid**: Resize icons and columns to fit your preference.
- **Custom Tabs**: Organize your emotes into manageable groups.
- **Locked Emotes Filter**: Option to hide or highlight emotes you haven't unlocked yet.
- **Performance Optimized**: Low-overhead ImGui implementation.
